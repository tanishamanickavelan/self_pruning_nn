Theory & Intuition
Why L1 on Sigmoid Gates Works

🚪 The Gate Mechanism

gates = σ(gate_scores) ∈ (0, 1)
pruned_w = weight × gates
output = x @ pruned_w.T + bias

Each weight gets a learnable scalar "gate score". After applying the Sigmoid function, the gate is bounded between 0 and 1. A gate near 0 nullifies its weight — effectively pruning it. Gradients flow through both weight and gate_scores via the chain rule.

📉 Why L1 Induces Sparsity

L_total = CrossEntropy + λ × Σ gate_i
∂L/∂gate_i = λ × sign(gate_i) = +λ
(gates are always positive after sigmoid)

The L1 penalty contributes a constant gradient of −λ per gate, regardless of its magnitude. Unlike L2, this steady push never "gives up" on small values — it drives them all the way to zero, producing true sparsity. This is the same principle that makes LASSO regression prefer exact zeros.

🎛️ The Role of Lambda (λ)

Lambda controls the sparsity-accuracy trade-off. A higher λ means the network is more strongly penalized for having active gates — resulting in more aggressive pruning. A lower λ allows the network to focus mainly on accuracy, keeping most weights alive. The best λ depends on your deployment constraints (memory budget vs. acceptable accuracy loss).

📊 Gate Distribution After Training

A successfully trained self-pruning network shows a bimodal gate distribution: a large spike near 0 (pruned weights) and a smaller cluster near 0.5–1.0 (important active weights). This clean separation shows the network has learned a meaningful binary mask for its own connections.

Key Implementation
PrunableLinear — Core Code
src/train.py · PrunableLinear 
class PrunableLinear(nn.Module):
    """
    Drop-in replacement for nn.Linear with learnable gate parameters.
    gate = sigmoid(gate_scores)  ∈ (0,1)
    pruned_weight = weight × gate      ← element-wise
    output = F.linear(x, pruned_weight, bias)
    """

    def __init__(self, in_features, out_features):
        super().__init__()
        # Standard weight & bias (Kaiming uniform init)
        self.weight      = nn.Parameter(torch.empty(out_features, in_features))
        self.bias        = nn.Parameter(torch.zeros(out_features))
        nn.init.kaiming_uniform_(self.weight, a=math.sqrt(5))

        # Learnable gate scores — same shape as weight
        # init at 0  →  sigmoid(0) = 0.5  (all gates start at 50%)
        self.gate_scores = nn.Parameter(torch.zeros(out_features, in_features))

    def get_gates(self):
        return torch.sigmoid(self.gate_scores)   # ∈ (0,1)

    def forward(self, x):
        gates          = self.get_gates()
        pruned_weights = self.weight * gates        # element-wise mask
        return F.linear(x, pruned_weights, self.bias)

# ─── Sparsity loss (L1 on all gates) ────────────────────────────
def sparsity_loss(model):
    return sum(
        layer.get_gates().sum()
        for layer in model.modules()
        if isinstance(layer, PrunableLinear)
    )

# ─── Training step ──────────────────────────────────────────────
logits    = model(images)
clf_loss  = F.cross_entropy(logits, labels)
spar_loss = sparsity_loss(model)
loss      = clf_loss + lam * spar_loss   # Total Loss = CLF + λ·L1
loss.backward()
optimizer.step()
